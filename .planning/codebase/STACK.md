# Tech Stack

## Core
- **Language**: TypeScript
- **Framework**: React Native with Expo (v54.0.27)
- **Web Rendering**: React Native Web (v0.21.0)
- **Runtime**: React 19.1.0, RN 0.81.5

## UI & Design
- **Icons**: Lucide (via @expo/vector-icons Feather)
- **Styling**: Vanilla RN StyleSheet with glassmorphism/SoftUI patterns
- **Animations**: React Native Reanimated (v4.1.1)
- **Feedback**: Expo Haptics

## State & Data
- **State Management**: React Context API (Custom AppProvider)
- **Data Fetching**: TanStack React Query (v5.90.21)
- **Database/Auth**: Supabase (@supabase/supabase-js v2.49.1)
- **Validation**: Zod (v3.25.7)

## Environment & Tooling
- **Package Manager**: pnpm (Monorepo)
- **Error Tracking**: Sentry (@sentry/react-native v8.5.0)
- **Navigation**: Expo Router (v6.0.17)
